Introduction

UI is fun to work with because something can easily be creative but still be easy to understand as the user. That is what I am to do, make UI professional, but fun. My process in doing things is to make something I think is fun and then start working downwards to make it functional and easy for people to understand. This logic usually applies to games, but I do my best to transfer this idea into web design.

Below are current and past projects that I have worked on.

FIGMA

Elysium Site

Project Information

UI/UX, FIGMA

This was done for a class project to display all of my past works featuring Elysium in an interesting way using Figma.

The target for this is to allow the users to see all of the games that features this character and then allow the user to get more information about each game. I work to include ways to contact myself and my socials that might be relevant in the production. Users may also see what platform each game is made on and a link to download.

Drum Duels

Project Information

Unreal, Art (3D Assets), UI/UX, Level Design

Drum Duels is an accessible game made in a team of three. This game was for a client who was blind.

Besides level design, I took careful consideration on the UI. Because my client was blind and can only see very little, I knew that I would have to use some high contrasting colors so she would still know her way when sound cues wasn't an option. I designed the levels to be in a garage where some bands start up in movies or video games, so the UI was catered towards that and feeling flashy.

IdaGem

Project Information

Unity, Art, UI/UX

A educational website and unity application that is going being used in Idaho schools by GIMM Works. It is supposed to teach about Idaho's history.

While making the avatars was my main job, I also had the pleasure of working with the UI and other various objects that appear around the world. My UI are usually literal interpretations of what the other team members suggested due to the fact that I tend to go overboard when they want somthing simple. Despite that, I still did my best to give all UI style.

Boise Quest

Project Information

Unity, AR, UI/UX, Art (3D Assets)

This application is an add-on to a project with a similar name for the City of Boise through GIMM Works. It's to encourage people to go to their local parks and also learn the history of the parks.

I was in charge of making the new UI so I did my best to be creative, but still favor readability and convenience. I wanted to tap into an earthy tree feel due to the fact that we were honoring the parks and plan to have a tree character as our guide. When finding a way to store all of the information, it only felt right if we had a log or journal.
