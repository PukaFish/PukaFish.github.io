Genshin Impact Themed Websites

Project Information

Throughout my web development classes, I have been working on passion project for my favorite game, Genshin Impact. When it comes to web design UI, it takes a lot more effort and testing to make things the way you want it to look. However, although these are simple looking websites, their best part is in the backend with all the funcitonality they can do. The backend uses Amazon Web Services and MySQL alongside Node.js. Because of these, the sites have functionality to sort, delete, and create.

Below are more information about each game.

Genshin Character Creation Website

HTML, CSS, Javascript, AWS, MySQL, Node.js

The Genshin Impact Character Creation Website is a website that allows users to input information about their OC (original character) and store it in a database with their necessary information. This information is stored within a AWS server and is referenced using MySQL. This website has some bad UI design, but the class had a requirement of a lot of CSS, so it is a little over the top.

Genshin Character Survey

HTML, CSS, Javascript, AWS, MySQL, Node.js

The Genshin Impact Character Survey website is a website that collects users' input in little cards to the side. It displays the information back by taking the information with MySQL from the AWS database. Has fairly simple UI, but it works.

Genshin Popularity Website

HTML, CSS, AWS, MySQL, D3.js, Javascript, Node.js

This website is the first of the series where it was my first time making a website with backend development adn D3.js charts. It was a struggle, but I am proud of how it turned out. Everything looks decently good and it is able to sort and display data in a meaningful way.
